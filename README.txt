A simple, light-weight plugin that allows researchers to dynamically display articles from Pub Med.

== Description ==

As an alternative to having to manage publication content as WordPress post content, this plugin will dynamically pull publications authored by the researchers entered in the backend. 



== Installation ==


1. Upload `pub_med_query.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Control settings in the Pub Med page section
4. Use shortcode [pub_med_query] on any page or post to display publications
